Create `popup/popup.js` as an ES module that imports from `../lib/storage.js`.

**Exported functions** (for testability — export handler functions so popup.test.js can import and test them independently):

1. `initPopup()` — called on DOMContentLoaded:
   - `const [tab] = await chrome.tabs.query({ active: true, currentWindow: true })`
   - Try `hostname = new URL(tab.url).hostname` — catch: show not-available state and return
   - `const config = await getDomainConfig(hostname) ?? { enabled: false, selectors: [], loadDelay: 0 }`
   - Store `hostname`, `config`, `tabId` in module-level variables
   - Call `renderPopup(hostname, config, tab.id)`

2. `renderPopup(hostname, config, tabId)` — renders full popup state:
   - Set `#domain-name` textContent to hostname
   - Set `#master-toggle` checked to config.enabled
   - Apply/remove `selector-list--disabled` class on `#selector-list` based on config.enabled
   - Clear `#selector-list` children
   - If config.selectors.length === 0: show `#empty-state`, hide selector rows
   - Else: hide `#empty-state`, render each selector via `renderSelectorRow()`

3. `renderSelectorRow(sel, index, config, hostname, tabId)` — creates one row:
   - `<div class="selector-row">`
   - `<span class="selector-text" title="${sel.selector}">${sel.selector}</span>`
   - `<input type="checkbox" checked=${sel.enabled}>` — onChange: `config.selectors[index].enabled = checkbox.checked; await setDomainConfig(hostname, config);`
   - `<button class="delete-btn" title="Remove selector" aria-label="Remove selector">✕</button>` — onClick: `config.selectors.splice(index, 1); await setDomainConfig(hostname, config); renderPopup(hostname, config, tabId);`
   - mouseenter and mouseleave handlers will be added in Plan 02 (not in this plan)
   - Return the row element; append to `#selector-list`

4. **Master toggle handler** — `#master-toggle` change event:
   - `config.enabled = toggle.checked`
   - `await setDomainConfig(hostname, config)`
   - Apply/remove `selector-list--disabled` class

5. **Add Selector handler** — `#add-selector-btn` click:
   - Show `#add-selector-msg` (remove hidden attribute)
   - `setTimeout(() => { addMsg.hidden = true; }, 3000)` — auto-hide after 3 seconds

6. **DOMContentLoaded listener:**
   ```javascript
   document.addEventListener('DOMContentLoaded', initPopup);
   ```
