Create `lib/picker.js` as an ES module that also sets `window._hrtlPicker` for use by content.js (a plain script). The file exports for Jest and assigns to window for content script consumption.

Structure of lib/picker.js:

1. **State enum:**
```javascript
const PICKER_STATE = { INACTIVE: 'INACTIVE', HOVERING: 'HOVERING', ELEMENT_SELECTED: 'ELEMENT_SELECTED', SAVING: 'SAVING' };
```

2. **Module-level state variables:**
- `_pickerState` (default: PICKER_STATE.INACTIVE)
- `_pickerHostname` (null)
- `_pickerHoveredEl` (null)
- `_pickerSelectedEl` (null)
- `_pickerSavedOutline` ('')
- `_pickerSavedOutlineOffset` ('')
- `_pickerBannerHost` (null)
- `_pickerOverlayHost` (null)
- `_pickerTooltipHost` (null)
- `_pickerMoveTimer` (null)

3. **generateSelector(el):**
- If `el.id` exists, return `#${CSS.escape(el.id)}`
- Else build `tag` + `.class1.class2` (each class CSS.escape'd)
- If no classes, return just tag (lowercased)

4. **buildAncestorChain(el):**
- Walk from el up parentElement, stop at document.body (exclusive), max 10 entries
- Return array: [el, parent, grandparent, ...]

5. **ancestorLabel(el):**
- If el.id: `${el.tagName.toLowerCase()}#${el.id}`
- Else if el.classList.length: `${el.tagName.toLowerCase()}.${Array.from(el.classList).join('.')}`
- Else: `el.tagName.toLowerCase()`

6. **buildSelector(ancestor, enabledClasses, enabledAttrs):**
- Start with `ancestor.tagName.toLowerCase()`
- If ancestor.id: append `#${CSS.escape(ancestor.id)}`
- Append each enabledClass: `.${CSS.escape(cls)}`
- Append each enabledAttr: `[${attr}="${CSS.escape(value)}"]`
- Return the combined string

7. **injectPickerBanner():**
- Create div with id `hrtl-picker-banner-host`, append to document.body
- Attach open shadow root
- Inject `<style>` + banner div per 04-UI-SPEC.md:
  - Position fixed, top 0, left 0, right 0, height 40px
  - Background #FFFFFF, border-bottom 1px solid #E5E7EB, border-left 4px solid #F59E0B
  - Font 13px/1.4 system-ui, color #111827, z-index 2147483647
  - role="status" aria-live="polite"
  - Text: "Click an element to configure Hebrew RTL — Esc to cancel"
- Store reference in _pickerBannerHost

8. **injectPickerTooltip(el, clientX, clientY):**
- If _pickerTooltipHost does not exist, create div id `hrtl-picker-tooltip-host`, append to body, attachShadow open
- Inject `<style>`:
  - Position fixed, background #111827, color #FFFFFF, border-radius 4px, padding 4px 8px
  - Font-size 13px, max-width 320px, overflow hidden, text-overflow ellipsis, white-space nowrap
  - z-index 2147483646
- Content: line 1 = ancestorLabel(el), line 2 = generateSelector(el) (use `<br>` between)
- Position: calculate top from el.getBoundingClientRect() — 8px above element, or 8px below if near viewport top (rect.top < 50)
- Left: clientX, clamped so tooltip does not overflow viewport right edge

9. **removePickerTooltip():**
- Remove _pickerTooltipHost from DOM if exists, set to null

10. **applyPickerOutline(el):**
- Save el.style.outline and el.style.outlineOffset
- Set `outline: 2px solid #F59E0B; outline-offset: 2px`
- Store el in _pickerHoveredEl

11. **removePickerOutline():**
- Restore saved outline/outlineOffset on _pickerHoveredEl, set _pickerHoveredEl to null

12. **onPickerMouseMove(e):** (named function, not arrow)
- clearTimeout(_pickerMoveTimer)
- setTimeout 16ms:
  - composedPath = e.composedPath()
  - target = composedPath[0]
  - Skip if target is inside _pickerBannerHost or _pickerTooltipHost or _pickerOverlayHost (check composedPath.includes)
  - If target !== _pickerHoveredEl: removePickerOutline(), applyPickerOutline(target), injectPickerTooltip(target, e.clientX, e.clientY)

13. **onPickerClick(e):** (named function, capture phase)
- composedPath = e.composedPath()
- Allow if path includes _pickerBannerHost or _pickerOverlayHost: return without intercepting
- e.stopPropagation(); e.preventDefault()
- If _pickerState !== HOVERING: return
- target = composedPath[0]
- removePickerTooltip()
- Set _pickerSelectedEl = target
- Set _pickerState = ELEMENT_SELECTED
- Call `pickerOpenOverlay(target)` (this is a stub in Plan 01; Plan 02 fills it in. For now: just a no-op placeholder that logs to console)

14. **onPickerKeyDown(e):** (named function, capture phase)
- If e.key === 'Escape': pickerReset()

15. **pickerActivate(hostname):**
- Set _pickerState = HOVERING, _pickerHostname = hostname
- injectPickerBanner()
- document.body.style.cursor = 'crosshair'
- addEventListener('mousemove', onPickerMouseMove, { passive: true })
- addEventListener('click', onPickerClick, true)
- addEventListener('keydown', onPickerKeyDown, true)

16. **pickerReset():**
- Set _pickerState = INACTIVE
- removePickerBanner() (remove _pickerBannerHost from DOM, null it)
- removePickerTooltip()
- removePickerOverlay() (remove _pickerOverlayHost from DOM, null it — will be populated by Plan 02)
- removePickerOutline()
- document.body.style.cursor = ''
- removeEventListener('mousemove', onPickerMouseMove)
- removeEventListener('click', onPickerClick, true)
- removeEventListener('keydown', onPickerKeyDown, true)
- clearTimeout(_pickerMoveTimer)
- _pickerSelectedEl = null
- _pickerHostname = null

17. **pickerOpenOverlay(target):** (placeholder for Plan 02)
- console.log('[Hebrew RTL] Selector Builder will open here for:', generateSelector(target))
- This function will be replaced in Plan 02 with the full Selector Builder overlay

18. **getPickerState():** Returns _pickerState (for testing)

19. **Exports + window assignment:**

export { PICKER_STATE, generateSelector, buildAncestorChain, ancestorLabel, buildSelector, pickerActivate, pickerReset, getPickerState, pickerOpenOverlay };
